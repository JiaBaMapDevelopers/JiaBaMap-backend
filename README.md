## Install

```sh
$ npm install
```

## Usage

1. Create an .env file with your Google API key
```
API_KEY = <your API key>
```

2. Generate the api document JSON first
```sh
$ npm run swagger
```

3. Run the dev server
```sh
$ npm run dev
```

## API Document
Please go to `http://localhost:3000/api-docs`
to interact with the API document.
