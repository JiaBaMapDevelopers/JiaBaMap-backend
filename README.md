## Install

1.

```sh
$ npm install
```

2. Install and launch MongoDB in your machine

- [macOS](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/)
- [Windows](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-windows/)
